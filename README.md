# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

# 🛍️ BRAVE – Premium Fashion E-Commerce Platform

BRAVE is a modern, production-ready fashion e-commerce platform developed for **Sai Deepthi Dresses**. The application delivers a premium online shopping experience with elegant UI, responsive design, secure authentication, shopping cart management, wishlist functionality, and an integrated checkout flow.

## ✨ Features

### Customer Experience

* Premium responsive homepage
* Three homepage design variants (Kombai, Cinematic, Classic)
* Product catalog with category browsing
* Advanced search, filtering, sorting, and pagination
* Product detail pages
* Wishlist functionality
* Shopping cart with live updates
* Multi-step checkout experience
* Razorpay payment integration
* Smooth page transitions and animations
* WhatsApp contact integration

### Authentication

* Clerk Authentication
* Firebase Authentication
* Google Sign-In
* Phone OTP (Firebase)
* User profile management

### State Management

* Authentication Context
* Cart Context
* Wishlist Context

### Performance & UX

* React Context API
* Framer Motion animations
* Lenis smooth scrolling
* Responsive layouts
* Optimized component architecture

## 🚀 Project Status

The customer-facing e-commerce platform is largely complete and includes the core shopping experience, authentication, and payment integration. Future enhancements include a comprehensive admin dashboard, inventory management, order management, and additional backend services.

## 🎯 Objective

To provide Sai Deepthi Dresses with a scalable, modern, and visually engaging e-commerce platform that enhances online shopping, improves customer experience, and supports future business growth.
